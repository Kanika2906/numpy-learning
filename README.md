# NumPy Learning

A collection of my **NumPy learning notes**, practice files, and beginner-friendly projects built while learning numerical computing with Python.

## 📚 Repository Contents

### 📝 NumPy Notes

* Well-organized NumPy notes
* Practice examples
* Important concepts and syntax
* Useful operations for quick revision

### 🚀 Projects

#### 1. General Questions

A collection of NumPy-based programming questions and solutions to strengthen problem-solving skills and understand array manipulation techniques.

**Topics Covered**

* Array creation
* Indexing & slicing
* Mathematical operations
* Logical operations
* Searching & sorting
* Matrix operations

---

#### 2. Valid Sudoku

A Sudoku validator built using **NumPy** that checks whether a 9×9 Sudoku board is valid by verifying:

* Every row
* Every column
* Every 3×3 sub-grid

---

#### 3. Tic Tac Toe

A simple Tic Tac Toe game built with **Streamlit** and **NumPy**.

**Features**

* Interactive web interface
* Win detection
* Draw detection
* Clean and simple gameplay
